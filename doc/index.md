Getting Started
===============

Prerequisites
-------------

This library require:

- jquery
- select2
- fxp-jquery-scroller

Installation
------------

```
npm install @fxp/select2-scroller --save
```

Dev installation
----------------

### Use NPM

```
npm install
```

### Use grunt

```
grunt serve
```

For ajax example, you must run this command in parallel:

```
grunt nodemon
```
